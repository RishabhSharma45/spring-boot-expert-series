# ⚡ Top 20 Spring Boot Development Coding/Design Questions – Level 2

### 1. One-to-Many Mapping

Department aur Employee entity banao (ek department ke kai employees). REST API likho.

### 2. Many-to-Many Mapping

Student aur Course ka relationship banao jisme ek student multiple courses me ho sakta hai aur vice versa.

### 3. Custom Query with @Query

Repository me @Query ka use karke native SQL likho (jaise salary > 50,000 employees nikaalo).

### 4. Specification / Criteria API

Dynamic search likho using Spring Data JPA Specifications.

### 5. ResponseEntity Usage

Controller API likho jo ResponseEntity ke sath status code aur body return kare.

### 6. Custom Exception + Global Error Response Format

Exception class aur ek ErrorDetails response object banao jo har API failure me common return ho.

### 7. Spring Boot Profiles ❗

application-dev.properties aur application-prod.properties ka use karke environment wise config dikhao.

### 8. CommandLineRunner

Ek demo likho jisme application start hone par DB me ek default user save ho.

### 9. Spring Boot Events 

Custom event create karke ek listener class likho (ex: user registration hone par event trigger ho).

### 10. Spring Boot REST Template / WebClient ❗

Dusre API ko call karne ka example likho.

### 11. JWT Authentication ❗

Login API likho jo JWT token generate kare aur dusre secured APIs token ke sath hi access ho.

### 12. Spring Boot Caching ❗

@Cacheable, @CacheEvict ka example likho (Ex: product list cache karna).

### 13. Spring Boot Interceptor / Filter ❗

Request aane se pehle ek filter lagao jo request ko log kare.

### 14. Custom Annotations ❗

Ek apni annotation banao (ex: @LogExecutionTime) jo method ka execution time calculate kare.

### 15. Spring Boot Async Programming ❗

@Async ka use karke ek API likho jo background thread me chale.

### 16. Spring Boot File Download API ❗

Upload ke baad ek download endpoint likho jo file return kare.

### 17. Multipart + JSON Payload ❗

Ek API likho jisme ek JSON data aur ek file dono ek sath bhej sake.

### 18. Spring Boot with Swagger (OpenAPI) ❗

API documentation dikhane ke liye Swagger config likho.

### 19. Spring Boot with H2 Database ❗

In-memory DB use karke CRUD app likho.

### 20. Unit Testing with @SpringBootTest + Mockito ❗

Service layer ka ek unit test likho mocking repository ke sath.
